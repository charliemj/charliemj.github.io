StitchHub

Final Project for MIT's 6.170: Software Studio

Check out StitchHub a final project for a software/web development class I took at MIT during fall 2016. I was a member of a four-person team, National Badger, that made StitchHub, a web app to help people make and share knitting/stitching/etc patterns. We took StitchHub from concepts to implementation in about six weeks. You can see the latest prototype here! You can see the code on my github, but I mostly contributed to StitchHub's backend using Node.js, Express.js, and MongoDB, and its UI design where I was in charge of most of the CSS and I made the inital UI/UX wireframes we based the project off of.
